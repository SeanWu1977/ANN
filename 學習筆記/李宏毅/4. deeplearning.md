# 矩陣
```
x1 — n1 —> y1 => active function(ex. sigmod) => new_x1
   X
x2 — n2 —> y2 => active function(ex. sigmod) => new_x2

w1 ==> for x1
w2 ==> for x2

n1 = [n1w1, n1w2]
n2 = [n2w1, n2w2]
n= [n1, n2]
x = [x1, x2]
b = [b1, b2]
n * x + b = y

y1 = n1w1*x1 + n1w2*x2 + b1
y2 = n2w1*x1 + n2w2*x2 + b2
```
