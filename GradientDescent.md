* 最簡單的loss function = Σ(y-yp)^2
* 要取最小，即預測值跟原值差最少
假設 yp = xw

▽=dL/dw == 算出斜率
當dL/dw > 0，則 w 要減少
當dL/dw < 0，則 w 要增加
dL/dw = 0，完美(但可能不是最低local optimal，有雙谷，但liner 不會有雙谷)

η == 學習率

w1 = w0 - η*(dL/dw0)
此處用(-)是因為斜率(dL/dw0)是(-)要增加w0
反之亦然。

此處也有斜率越陡，調整越多的效果。
